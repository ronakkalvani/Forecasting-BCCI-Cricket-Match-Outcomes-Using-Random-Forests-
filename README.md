# Forecasting-BCCI-Cricket-Match-Outcomes-Using-Random-Forests-

Developed decision trees using mutual information as a attribute selector, Implemented one hot encoding to get more information from categorical attributes and two-way split from median attribute value for continuous Features

Analysed validation set accuracy over different tree depths after post pruning tree, Extended analysis to Random Forests using grid search over n estimators, max features and min samples split to get out of bag accuracy of 73%
